Renderização SPA 

  O servidor requisita a url e recebe um arquivo html com a referencia de um arquivo js.
  Quando esse arquivo chega no computatdor ele começa a controlar o dom, montando a aplicação
  dentro do navegador no lado cliente, os processos de atualização de dados são executados de
  forma automática sem que cliente precise requisitar manualmente.     
          

Renderização SSR 

  Server side Rendering o servidor compila a view da aplicação junto com os dados e envia 
  para cada cliente que pedir com base nos parâmetros de cada requisição, esse tipo
  de renderização normalmente é utilizada por plataformas ead e blogs.
    

Renderização SSG

  SSG retorna arquivos que já estão salvos dentro do servidor são arquivos estáticos que o
  cliente faz uma requisição direto para a url do diretorio onde os arquivos foram
  salvos.
